# micronotes

micronotes is a small, fast, Linux-only Markdown notes app built with C++ and SDL3.

This repo is 100% vibecoded: every source file, test, and document here was written by AI
coding agents under human direction. It is a real app, not a fake demo, but it is still a
work-in-progress and should be read with that in mind.

See [docs/build.md](docs/build.md) for the required Linux toolchain, SDL3-from-source setup, and local build commands.

See [docs/library-format.md](docs/library-format.md) for the local library format, Markdown scope, and attachment behavior.

See [docs/markdown-elements.md](docs/markdown-elements.md) for a Markdown renderer fixture that doubles as documentation for supported elements.
